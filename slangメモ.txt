■playground用機能
https://shader-slang.org/slang-playground/

outputがすべてがでるわけではなく、採用されるスレッドがランダムなので
printがあまり役に立たない


[playground::CALL(1, 1, 1)]
呼び出すグリッドの数。画面全部を呼ぶことはplaygroundではなぜかできない。
（下のSIZE_OFとどちらかは必須っぽい）

[playground::CALL::SIZE_OF("outputTexture")]
出力領域のサイズ分 computeを呼び出す
（playgroundで画面全部を呼びたいときはこれを使うしかない？）


[playground::CALL::ONCE]
スレッドが単一になるわけではないので、1つの結果を見たいだけならあったほうが良い
----------------------------------------------------------------------------------------
// 基本的なコード
import playground;
import rendering;

[shader("compute")]
// playgroundで画面全体分呼び出すにはこれしかない
[playground::CALL::SIZE_OF("outputTexture")]
// 出力は１回でよい
[playground::CALL::ONCE]
void imageMain(uint2 dispatchThreadID: SV_DispatchThreadID)
{
	// dispatchThreadIDはgroupIDに関わらない
    drawPixel(dispatchThreadID, (int2 _) => float4(0.3, 0.7, 0.55, 1.0));
}
